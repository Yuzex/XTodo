# XTodo v1.0.0 — 首个正式版本

> 悬浮置顶、随用随走的 Windows 桌面待办清单。

---

## 🎯 核心亮点

- **悬浮置顶**，始终在最上层，不遮挡时自动收起为一行窄条
- **三层任务层级**，根任务 → 子任务 → 孙任务
- **多分类 Tab**，拖拽排序、双击重命名、右键删除
- **闭环条件**，文字完成标准 + 可勾选子检查项
- **系统托盘驻留**，不在任务栏显示，关闭窗口即隐藏到托盘
- **开机自启动**，设置中一键开关
- **独立自包含发布**，无需安装 .NET 运行时，解压即用

---

## 🚀 使用方式

1. 下载下方 `XTodo_v1.0.0.zip`
2. 解压到任意目录
3. 双击 `XTodo.exe` 即可运行

> 关闭窗口后程序隐藏到系统托盘，不会退出。右键托盘图标 → **退出** 可彻底关闭。

---

## 🖱 操作速览

| 操作 | 方式 |
|------|------|
| 勾选完成 | 点击左侧勾选框 |
| 重命名任务 | 双击任务名 |
| 添加子任务 | 鼠标悬停 → 点击 **＋** |
| 删除任务 | 鼠标悬停 → 点击 **✕** |
| 新建分类 | Tab 栏左侧 **➕** |
| 重命名分类 | 双击 Tab |
| 删除分类 | 右键 Tab |
| 调整分类顺序 | 拖拽 Tab |
| 设置完成时间 | 点击 ⏱ 按钮 |
| 编辑完成标准 | 点击 📝 按钮 |
| 标记当前进行中的任务 | 点击 ★ / ☆ |
| 手动保存 | `Ctrl + S` |
| 开机自启 | ⚙ → 开机自启动 |
