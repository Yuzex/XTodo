# XTodo — 用户需求规格说明

> 本文档使用非技术语言描述软件功能需求，供用户阅读和确认。

---

## 1. 软件定位

一个悬浮在 Windows 桌面上的 Todo List 工具，帮助你管理日常任务。它会始终悬浮在屏幕顶部，不用时可自动收起，不占桌面空间。

---

## 2. 功能需求

### 2.1 任务管理

每条任务包含以下信息：

| 内容 | 说明 | 是否必填 |
|------|------|----------|
| 任务名称 | 任务的标题 | ✅ 必填 |
| 完成状态 | ☐ 待办 / ☑ 已完成 | 自动 |
| 闭环条件（文字） | 一段话描述"怎样算完成" | 选填 |
| 闭环条件（子检查项） | 多个可勾选的小步骤 | 选填 |
| 预计完成时间 | 截止日期和时间 | 选填 |

任务支持**层级结构**：
- 根任务（一级）
  - 子任务（二级）
    - 孙任务（三级）
- 最多 3 层

### 2.2 窗口外观与行为

#### 展开态（鼠标移入时）
- 窗口位于**屏幕顶部居中**
- 宽度约**屏幕宽度的 1/3**
- 显示完整任务列表和 Tab 分类
- 可**拖拽窗口边缘**调整大小

#### 收起态（鼠标移出时）
- 折叠为一行窄条
- 仅显示三样信息：`待办数量 | 当前根任务名称 | 预计完成时间`
- 当前根任务的选择规则：
  1. 优先显示**手动标记**为"进行中"的根任务
  2. 如果没有手动标记，则显示**截止时间最近**的未完成根任务

### 2.3 分类管理

- 使用**Tab 页签**对任务分类（如"工作"、"个人"、"学习"）
- 支持**新建、重命名、删除**分类
- 每个分类内的任务是独立的

### 2.4 数据存储

- 数据自动保存在本地文件中
- 关闭软件后数据不会丢失
- 下次打开自动恢复

---

## 3. 界面风格

- **风格**：简洁直观，不花哨
- **主色调**：淡黄色（类似便签纸的温暖色调）
- **完成态任务**：文字加删除线，颜色变灰
- **字体**：清晰易读的系统中文字体

---

## 4. 非功能需求

- 软件运行在 Windows 10/11 上
- 不需要安装，双击即可运行
- 窗口始终悬浮在其它窗口上方
- 操作响应流畅
