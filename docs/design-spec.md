# XTodo — UI 设计规范

> 本文档定义所有 UI 相关的常量、配色、字体、间距和交互行为，确保视觉一致性。

---

## 1. 色彩系统

### 主色调（淡黄色系）

| 用途 | 色值 | 说明 |
|------|------|------|
| 面板背景 | `#FFF8E1` | 超浅黄，主面板底色 |
| Tab 选中态 | `#FFECB3` | 浅琥珀色，当前选中的 Tab |
| Tab 未选中 | `#FFF3E0` | 略深的浅色，未选中 Tab |
| 按钮悬浮 | `#FFE082` | 中等琥珀，鼠标悬停反馈 |
| 边框线 | `#E0D7C2` | 暖灰边框，分隔线用 |

### 文字颜色

| 用途 | 色值 | 说明 |
|------|------|------|
| 正文 | `#3E2723` | 深棕色，主文字色 |
| 次要文字 | `#8D6E63` | 暖灰棕，提示信息 |
| 完成态文字 | `#BDBDBD` | 浅灰，完成后的划线文字 |
| 收起态文字 | `#5D4037` | 中棕色 |

### 功能色

| 用途 | 色值 | 说明 |
|------|------|------|
| 添加按钮 | `#81C784` | 柔绿，新增操作 |
| 删除按钮 | `#E57373` | 柔红，删除操作 |
| 标记进行中 | `#FFB74D` | 柔橙，当前任务标记 |

## 2. 字体

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| 收起态 | 13px | Normal | 窄条上的摘要文字 |
| 根任务 | 15px | SemiBold | 一级任务名 |
| 子任务 | 14px | Normal | 二级任务名 |
| 孙任务 | 13px | Normal | 三级任务名 |
| 辅助信息 | 11px | Normal | 闭环条件、时间等 |

- **字体族**：`Microsoft YaHei`（微软雅黑，Windows 原生中文字体）
- 回退字体：`Segoe UI, sans-serif`

## 3. 尺寸与间距

| 元素 | 尺寸 | 说明 |
|------|------|------|
| 收起态高度 | 40px | 窄条总高度 |
| 展开态最小高度 | 300px | 面板最小高度 |
| 展开态默认高度 | 500px | 面板默认高度 |
| 展开态默认宽度 | 屏幕宽度的 1/3 | 约 640px（1920 屏幕） |
| 展开态最小宽度 | 300px | 拖拽下限 |
| Tab 高度 | 36px | 页签栏高度 |
| 任务行高度 | 32px | 单行任务高度 |
| 层级缩进 | 24px | 每层向右缩进 |
| 面板内边距 | 12px | 内容区 padding |
| 圆角半径 | 8px | 面板外框圆角 |

## 4. 布局结构（展开态）

```
┌─────────────────────────────────────────────┐
│  [+ 新建分类]  [工作 Tab] [个人 Tab] [...]   │  ← Tab 栏 (36px)
├─────────────────────────────────────────────┤
│  [+ 添加根任务]                              │  ← 操作栏
│                                             │
│  ☐ 根任务 1          ⏱ 6/27   ★ 进行中     │  ← 任务行 (32px)
│    ☐ 子任务 1-1        ⏱ 6/28             │    缩进 24px
│      ☐ 孙任务 1-1-1    ⏱ 6/29           │    缩进 48px
│    ☑ 子任务 1-2 (已完) ~~文字划线~~        │    完成态
│  ☐ 根任务 2                                │
│                                             │
├─────────────────────────────────────────────┤
│ 拖拽调整手柄                                 │  ← 底部拖拽区
└─────────────────────────────────────────────┘
```

## 5. 布局结构（收起态）

```
│  3 项待办 │ 📋 优化首页性能 │ ⏱ 本周五      │  ← 单行 40px
└─────────────────────────────────────────────┘
```

## 6. 交互规范

| 操作 | 行为 | 说明 |
|------|------|------|
| 鼠标移入面板 | 300ms 从收起态展开 | 过渡动画 |
| 鼠标移出面板 | 500ms 延迟后收起 | 防止误触 |
| 点击勾选框 | 即时切换完成状态 | 无需确认 |
| 点击删除 | 弹出确认提示 | 防止误删 |
| 双击任务名 | 进入编辑模式 | 修改名称 |
| 拖拽边缘 | 连续调整宽/高 | 无级缩放 |
| 右键根任务 | 弹出菜单：标记为进行中 | 手动选择当前任务 |

## 7. 图标

优先使用简单 Unicode 字符代替图标文件：

| 图标 | Unicode | 用途 |
|------|---------|------|
| ☐ | U+2610 | 待办 |
| ☑ | U+2611 | 已完成 |
| ⏱ | U+23F1 | 预计时间 |
| ★ | U+2605 | 当前进行中标记 |
| ➕ | U+2795 | 添加 |
| ➖ | U+2796 | 删除 / 折叠 |
| ▶ | U+25B6 | 展开子任务 |
| ▼ | U+25BC | 折叠子任务 |
