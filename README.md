# XTodo — 悬浮待办清单

> 一个轻量、快捷的 Windows 桌面待办工具，悬浮在屏幕顶部，随用随走。

---

## ✨ 功能

- **悬浮置顶** — 始终在屏幕最上层，不遮挡工作区时可自动收起为窄条
- **层级任务** — 支持 根任务 → 子任务 → 孙任务 三层结构
- **多分类 Tab** — 按场景分标签管理，支持拖拽排序、双击重命名
- **闭环条件** — 为任务设定文字完成标准 + 可勾选子检查项
- **系统托盘** — 不在任务栏显示，关闭后最小化到托盘图标区
- **开机自启** — 设置中一键开启
- **本地存储** — 数据保存在 `%APPDATA%\XTodo\data.json`，离线可用，无隐私顾虑
- **免安装运行** — 发布包为独立自包含 exe，无需安装 .NET 运行时

---

## 🚀 快速开始

1. 从 `publish/` 目录获取 `XTodo.exe`
2. 双击运行
3. 首次启动自动创建示例数据，可直接上手体验

> 关闭窗口（✕ 或 Alt+F4）后程序会隐藏到系统托盘，不会退出。
> 右键点击托盘图标 → **退出** 可彻底关闭。

## 🖱 操作指南

| 操作 | 方式 |
|------|------|
| 勾选完成任务 | 点击任务左侧勾选框 |
| 重命名任务 | 双击任务名称 |
| 添加子任务 | 鼠标悬停任务行，点击右侧 **＋** |
| 删除任务 | 鼠标悬停任务行，点击右侧 **✕** |
| 新建分类 | 点击 Tab 栏左侧 **➕** |
| 重命名分类 | 双击 Tab 名称 |
| 删除分类 | 右键点击 Tab |
| 调整分类顺序 | 拖拽 Tab 到目标位置 |
| 手动保存 | `Ctrl + S` |
| 开机自启动 | 点击 ⚙ → 勾选“开机自启动” |

## 🛠 开发

### 技术栈

- .NET 10 + WPF + MVVM
- System.Text.Json（本地 JSON 持久化）
- Windows Forms（系统托盘集成）

### 构建

```powershell
# 还原依赖
dotnet restore

# 编译
dotnet build

# 发布为独立自包含 exe
dotnet publish -c Release -o ..\publish
```
